== Description ==

**WP Business Proposals** is the modern, secure, and simple way to manage business proposals right inside your WordPress dashboard.

- **Create, edit, and manage proposals** as a custom post type
- **Secure by design**: Follows WordPress Coding Standards and best practices
- **No bloat, no distractions**: Just what you need to get started
- **REST API ready**: Integrate with your favorite tools

> "I built this plugin because I believe every business deserves a simple, secure, and beautiful way to manage proposals—without leaving WordPress." — Lewis Ushindi, Creator

== Installation ==

1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. You will see a new 'Proposals' menu in your WordPress admin. Start creating proposals!

== Frequently Asked Questions ==

= Can I customize the proposal fields? =
Not in this version. The first release is intentionally simple. Custom fields and templates are on the roadmap.

= Is it secure? =
Absolutely. All output is sanitized and escaped. The plugin follows WordPress Coding Standards and best practices.

= Can I use this on client sites? =
Yes! It's lightweight, safe, and ready for production.
